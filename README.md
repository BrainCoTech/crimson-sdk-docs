# CrimsonSDK Docs

CrimsonSDK Docs

CrimsonSDK is a powerful, easy-to-use, and free SDK for creating EEG attention & meditation apps.

```shell
pnpm install
pnpm build
pnpm docs:dev

pnpm docs:build
open docs/.vuepress/dist
```
